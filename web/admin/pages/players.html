Players

A player related collections, such as details, info, avatars, outfits and items.

Details

The main GSFPlayer attributes for the selected player, including ActivePlayerAvatar, CreateDate for profile widget, IsTutorialCompleted for enabling social events, and IsQA used in ConsumableItemsManager.cs:102.
Other player fields, such as HomeThemeID, CurrentRaceMode, WorkshopOptions, YardBuildingID, LastLogin, PlayTime, HomeVillagePlotID, StoreVillagePlotID, PlayerStoreID, PlayerMazeID, and VillageID, are serialized by the client but do not appear to be used by game code.

Avatars

List of Zings/avatars available to the selected player. Each one has it's own name etc.

Outfits

Saved outfit presets for the selected player avatars. Each row links an outfit OID to a player avatar and outfit number.
