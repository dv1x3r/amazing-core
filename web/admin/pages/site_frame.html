Site Frame

A site-frame object contains the main asset container with core assets such as Player_Base, PlayerCamera, ShadersList and Avatar_SlotIds.txt.

The client requests this payload during intro loading and again after login. The only known type_value is 1.

Avatar_SlotIds.txt is a slot registry for wearable/equippable avatar parts. Also imported to the avatar_slot SQL table. Used in DressAvatarManager.cs:917

Main_Localization.txt is a localization file for buttons and main text.
