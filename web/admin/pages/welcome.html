Welcome to Amazing Core Dashboard!

This CMS dashboard lets you inspect and configure your Amazing Core server.

Assets

The Assets screen manages metadata stored in core.db: GSF OIDs, CDN IDs, asset type, group, and other fields.

Dummy Parameters

Dummy parameters are small overrides used for testing hard-coded values.

SQL Explorer and Asset Files

SQL Explorer gives direct access to core.db with a few example queries.
Asset Files manages blob.db, where uploaded cache files are stored for the built-in CDN.
