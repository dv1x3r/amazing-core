Welcome to the Amazing Core Dashboard!

This CMS dashboard allows you to inspect and configure your Amazing Core server.

Assets

Asset is a single resource file, such as Unity asset bundle, treenode, json, image, or audio file.
Asset Container is a group of assets. It can contain multiple assets and/or asset packages.
Asset Package is an asset container that is referenced by another asset container.

Collections

Items contain item collection.
Item Categories contain item categories definition.
Random Names contain random name parts for player name generation.
Site Frame contains the root asset container with core assets such as camera and shaders.
Zones contain zone scene containers.

World

Players manage player records, avatars, outfits, inventory items, and home/maze data.

Requests

Log Watcher provies a watcher to monitor and analyze game requests in real time.

SQL Explorer and Asset Files

The SQL Explorer provides direct access to core.db, along with a few example queries.
Asset Files manages blob.db, where uploaded cache files are stored for the built-in CDN.
