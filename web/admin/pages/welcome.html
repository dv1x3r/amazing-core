Welcome to Amazing Core Dashboard!


This is the admin dashboard for your Amazing Core server instance.
Use the sidebar to navigate between sections.


Assets

This section provides a view of known game assets and their associated types.
To edit `Asset Type`, `Asset Group`, `ResName`, or `Description`, double-click the cell.


Random Names

Manage the list of random name parts used during character creation.


SQL Explorer: core.db

Manage the core database with SQL.
There are some query examples available in the dropdown menu.


Cache Files: blob.db

Manage the blob database used for game cache asset storage and built-in CDN.
From this section you can upload individual files, import or export the whole `cache` directory.
