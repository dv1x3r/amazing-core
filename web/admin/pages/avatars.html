Avatars

An avatar represents a Zing model and its related assets.

Max Outfits controls whether dressing/outfits are enabled and how many outfit slots/presets are valid. If it is 0, the avatar is treated as not supporting saved outfits or clothing slots. If it is 1, only outfit slot 1 is valid. Higher values enable slots up to that number.

Other avatar fields, such as Dimensions, Weight, and Height, are serialized by the client but do not appear to be used by game code.
