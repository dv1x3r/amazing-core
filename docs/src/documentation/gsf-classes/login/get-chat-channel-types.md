# GetChatChannelTypes

## Description

Sent during the `ZoneTransitionQueue`.

Fetches the available chat channel type definitions.

## Request

No fields.

## Response

| Field              | Type                                                 | Description                |
| ------------------ | ---------------------------------------------------- | -------------------------- |
| `ChatChannelTypes` | [`[]ChatChannelType`](../types/chat-channel-type.md) | List of chat channel types |
